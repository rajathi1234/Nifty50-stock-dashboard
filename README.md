# Nifty50 Stock Dashboard

## Project Overview
This project analyzes Nifty50 stock data to provide insights on market trends.  
It downloads stock data, cleans and processes it, stores it in a database, performs analysis, and visualizes results using Python and Streamlit.

## Key CSV Files

- `analysis_outputs/nifty50_analysis.csv` — daily stock data with returns and cumulative returns.  
- `analysis_outputs/metrics_summary.csv` — summary statistics like total stocks, green/red stocks, best performer.  
- `analysis_outputs/metrics_by_symbol.csv` — per-stock metrics like average daily return and volatility.

## How to Run

Create and activate a virtual environment:

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate

Install dependencies:
pip install -r requirements.txt

Run scripts in order:
python scripts/01_download_all_stocks.py
python scripts/02_clean_data.py
python scripts/03_load_to_sql.py
python scripts/04_analysis.py
python scripts/05_visualize.py
python scripts/06_streamlit_app.py

Launch the Streamlit dashboard:
streamlit run scripts/06_streamlit_app.py

## Power BI Dashboard

This project includes a Power BI dashboard created using analysis outputs.

- Source file: `Nifty50_Stock_Analysis.pbix`
- Report export: `Nifty50_Stock_Analysis.pdf`

The dashboard contains:
- Summary metrics (Total stocks, Green/Red stocks, Best performer)
- Stock comparison charts
- Time-series analysis with filters

Note: Power BI files and PDF reports are excluded from GitHub using `.gitignore`.
